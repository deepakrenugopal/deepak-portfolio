AI Radiology Report Summariser

A T5-base model fine-tuned to automatically generate a clinical impression from chest X-ray findings. Trained on the Indiana University Chest X-Ray Collection (2,712 radiology reports), the model takes free-text findings and produces a concise clinical impression, achieving a ROUGE-1 score of 0.5525. The model is served from a HuggingFace Space and called via a FastAPI backend hosted on Render.

Tech Stack
  • Python, PyTorch, HuggingFace Transformers (T5-base)
  • FastAPI backend
  • Model hosted on HuggingFace Spaces
  • Deployed on Render

Visual Product Recommendations

A visual product recommendation system for a fashion retailer. A customer uploads a clothing image and the app returns the 5 most visually similar items from a catalogue of 31,378 images, each with a cosine similarity score. Image embeddings are generated using a pre-trained MobileNetV2 model, stored as float16 vectors, and compared at query time using cosine similarity. The MobileNetV2 model runs on a HuggingFace Space, called from a Flask backend on Render.

Tech Stack
  • Python, PyTorch, TorchVision (MobileNetV2)
  • Flask backend
  • Model hosted on HuggingFace Spaces, data on HuggingFace Datasets
  • Deployed on Render
